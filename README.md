If you want to measure the speed of a car or the angles of a spacecraft, you can easily use the BNO055 sensor.
Today we created a library for BNO055 to use with STM32. With this library you can configure BNO055 and retrieve data from it.

**WARNING:** This library is written on an STM32F407-DISC1 so its header file is `stm32f4xx_hal.h`. Ä°f you use another board, for example STM32F103C8 BluePill, just change that to `stm32f1xx_hal.h` in our BNO055_STM32.h header file.
```c
#include "stm32f1xx_hal.h
```
**WARNING:** We used the `printf()` function in this library. Therefore, you should activate the printf function. Just google it :)

Firstly, you must create a project with CubeMX and activate I2C protocol. Then add the header file `BNO055_STM32..h` into the `../Core/Inc` directory and the `BNO055.c` source file into the `../Core/Src` directory.

![github1](https://github.com/Afebia/Afebia/assets/121301223/84398f13-6241-46af-9364-4d6575963a09)

For change I2C address, go to `BNO055_STM32.h` header file. `0x28` or `0x29`
```c
#define P_BNO055 (0x28<<1)
```

Now, lets explain all functions. With this library, you can obtain Gyro, Accel, Magnet, Euler Angle, Linear Accel and Gravity Vector data. Call the relevant functions below to get the data you want.
```c
GyroData();
AccelData();
MagnetoData();
Euler_Angle();
Lineer_Accel();
Gravity_Vector();
```

You can change Axis remap of BNO055 with following function. 
```c
BNO055_Remap_Axis(axis_position_t position)
```
Replace `position` with one of the following. For detailed information, see the relevant section in the datasheet.
```c
typedef enum{  //BNO055 AXIS REMAP
	P0=0,
	P1,
	P2,
	P3,
	P4,
	P5,
	P6,
	P7
}axis_position_t;
```
Also you can set Accelerometer range with following function. For detailed information, see the relevant section in the datasheet.
```c
SET_Accel_Range(accel_range_t range)

typedef enum{ //ACCEL RANGES
	Range_2G=0x00,
	Range_4G=0x01,
	Range_8G=0x02,
	Range_16G=0x03
}accel_range_t;
```
Use the following function for initiliziation of BNO055. You can change the function according to your wishes.
```c
void BNO055_Init(void){

	SelectPage(Page_0);
	Set_Operation_Mode(CONFIG_MODE);

	uint8_t reset = 0x20;
	HAL_I2C_Mem_Write(&hi2c1, P_BNO055, P_SYS_TRIGGER, 1, &reset, sizeof(reset), HAL_MAX_DELAY);

	HAL_Delay(100);

	SetPowerTo_NORMAL();
	HAL_Delay(100);

	External_Crystal(true);
	HAL_Delay(100);

	Unit_Select(WINDOWS, TEMP_CELCIUS, EUL_DEG, GYRO_DPS, ACC_MS2);
	HAL_Delay(100);
}
```
